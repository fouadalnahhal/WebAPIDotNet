# WebAPIDotNet

Minimal ASP.NET Core Web API template (.NET 8+).

## Features
- RESTful endpoints
- Swagger UI
- Dependency injection
- CORS & global error handling
- EF Core ready (In-Memory by default)

## Quick Start

```bash
git clone https://github.com/fouadalnahhal/WebAPIDotNet.git
cd WebAPIDotNet
dotnet run
```

App runs at `https://localhost:7001`  
Swagger: `https://localhost:7001/swagger`

## Prerequisites
- .NET 8 SDK

## Commands
```bash
dotnet restore
dotnet build
dotnet run
dotnet test   # if tests exist
```
